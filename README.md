
# NearMe-APi

## Features
1. Login
2. Search for the closest placses 
3. Search for placses by the name
4. Search for placses by the catagorey
5. get the direction

## Stack
1. Runtime Environment: `Node.js`
2. Language: `Ts`
3. Web Server: `Fastify`
4. ORM: `prisma`
5. DB: `mongo`
